Ich interessiere mich für HTML & CSS

HTML steht für HyperText Markup Language. Mit HTML kann man Webseiten strukturieren. Man definiert welche Elemente angezeigt werden sollen und wo diese platziert werden. Die die Elemente tatsächlich aussehen, bestimmt man mit CSS.

Man spricht bei HTML nicht von einer Programmiersprache sondern von einer Auszeichnungsprache.
